Resources & Definitions

Key terms and concepts you'll encounter in your QA journey

Development Terminology

  • AGILE: Sprint-based development methodology
  • ARTs: Agile Release Trains
  • Sprints: 2-week development cycles
  • Stakeholders: Anyone invested in a feature

Testing Concepts

  • A/B Testing: Test version performance comparison
  • Smoke Test: < 1hr, core functionality validation
  • Regression Testing: Verify existing functionality
  • Sanity Testing: Quick verification of key features

Paramount Projects

  • Terra Nova: Migration to modern tech stack
  • Vodfather: Server-side playback project
  • Media Edge API: Content delivery system
  • AEP: Adobe Experience Platform integration
